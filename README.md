JavaScript Features Added (Task 3)
1. Contact Form Validation

JavaScript is used to validate the contact form fields:

Ensures the name field is not empty

Validates the email format using a regular expression

Ensures the message field is not empty

Prevents form submission if validation fails

2. Error Message Display

Displays clear error messages below each input field

Error messages are added and removed dynamically using DOM manipulation

3. Real-Time User Interaction

Error messages disappear automatically as the user starts typing

Implemented using input event listeners

4. Success Message on Submission

Displays a success message after successful form validation

Automatically clears the form after submission

5. Dark Mode Toggle

A dark mode toggle button is added to the navigation bar

Allows users to switch between light and dark themes

Improves accessibility and user experience

6. Theme Persistence Using LocalStorage

The selected theme (light or dark mode) is saved using localStorage

The preferred theme remains active even after refreshing the page
